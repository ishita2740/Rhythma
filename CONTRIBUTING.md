# Contributing to Rhythma AI 🌸

Thank you for your interest in contributing! Rhythma is an open-source project building AI-powered menstrual health tools for women in India, and every contribution — code, documentation, design, translations, or feedback — matters.

## How to Contribute

### 1. Fork and Clone

```bash
git clone https://github.com/ishita2740/Rhythma.git
cd Rhythma
```

### 2. Create a Branch

```bash
git checkout -b feature/your-feature-name
# or
git checkout -b fix/bug-description
```

### 3. Make Your Changes

- Follow the existing code style
- Keep commits focused and descriptive
- Add comments for complex logic

### 4. Test Your Changes

For Flutter:
```bash
flutter test
flutter analyze
```

For the backend:
```bash
pytest backend/
```

### 5. Submit a Pull Request

- Write a clear PR description explaining what you changed and why
- Reference any related issues with `Closes #issue_number`
- Screenshots are helpful for UI changes

## Areas Where Help Is Needed

| Area | What's needed |
|------|--------------|
| 🌍 **Translations** | Health content and UI strings in Hindi, Marathi, Tamil, Telugu, Kannada |
| 🤖 **ML Models** | CVI and MHS model training, evaluation, TFLite export |
| 🔧 **Backend** | FastAPI endpoints, Firestore schema, authentication |
| 🧪 **Testing** | Flutter widget tests, API integration tests |
| 📖 **Documentation** | API docs, setup guides, architecture diagrams |
| 🎨 **Design** | Accessibility improvements, new screen designs |
